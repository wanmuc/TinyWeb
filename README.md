# 1.TinyWeb
> 通epoll实现Reactor-MS模型，并在应用层支持http协议，实现一个小巧的http服务TinyWeb。

学习完本项目后，你将能锻炼到以下的技术能力，并加深对它们的理解。
- http协议格式
- http协议的编解码
- Reactor-MS模型

# 2.编译与运行
- 执行make命令编译，生成TinyWeb服务。
- 执行命令启动TinyWeb。
- 使用curl工具对TinyWeb做接口测试。
- 编译WebBench子目录的压测工具，使用WebBench对TinyWeb进行压测。
